3.what are the types of reaction ?

there are 6 types of reaction are - 
a  combination reaction: in this the many reactants form single product.
Example : 2MgO + O2 = 2MgO 
b  decomposition reactions: in this the single reactants form many product
Example : 2MgO = 2Mg + O2 
c  displacement reactions : in this the low reactive materials element exchange from high reactive materials element.
Example : Fe + CuSo4 = FeSo4 
d  double displacement reactions :in this reactions exchange their ions from their own compounds
Example : Na2So4 + BaCl2 = 2NaCl + BaSo4 
e  exothermic reactions =in which reactions which release heat after reacting
Example : CaCO3 = Cao + CO2 + heat 
f  electrolysis reactions : in this reactions which release electricity after reacting
Example : 2NaCl + 2H2O = 2NaOH + H2 
g  redox reactions : in this reation the both oxidation and reduction takes place in the same reaction
Example : ZnO + C = ZnO + CO
